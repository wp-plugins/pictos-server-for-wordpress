== Description ==
Pictos Server for WordPress makes it easy to use [Drew Wilson's Pictos Server Custom Fonts](http://pictos.cc/server) on your WordPress website. It saves your settings in theme mods, so you can use different accounts and combos for each theme in the case of a multisite installation.

= More Info =

[Pictos Server for WordPress](http://jeffsebring.com/wordpress/plugins/pictos-server-for-wordpress)

== Installation ==

1. Upload and activate Pictos Server for WordPress.
2. Navigate to the 'Pictos' menu under the 'Appearance' tab.
3. Enter your Pictos Server account number, and Combo number in the settings fields.
4. Save Changes
5. Use the font-family: "Pictos Custom" in your theme or child theme stylesheets to use your Pictos Custom Font.

== Frequently Asked Questions ==

= Where are my Pictos Account and Combo numbers? =

When you create your Pictos combo, look at the bottom of the page, and you will see a snippet of code similar to what I've provide below. You will find your account number in place of ACCOUNT_NUMBER, and the combo number in place of COMBO_NUMBER:

   <link href="//get.pictos.cc/fonts/ACCOUNT_NUMBER/COMBO_NUMBER" rel="stylesheet" type="text/css">
